# 🏎️ Race Visualizer System

A C++ project that simulates a 200m race using Data Structures and Algorithms.

## 🚀 Features
- Add Competitors
- Search by Name (Linear Search)
- Search by ID (Binary Search)
- Sort by Speed (Quick Sort)
- Sort by Distance (Merge Sort)
- Live Race Simulation
- Update Competitor Speed

## 🛠️ Technologies
- C++
- Data Structures
- Algorithms
